# NHL Prediction Model - Quick Start Guide

## TL;DR - Get Running in 5 Minutes

### Prerequisites
- Python 3.9+ installed
- `gamedata.csv` and `playergamedata.csv` files from assignment

### Step-by-Step

```bash
# 1. Create virtual environment
python -m venv nhl_env

# 2. Activate virtual environment
# On Windows:
nhl_env\Scripts\activate
# On Mac/Linux:
source nhl_env/bin/activate

# 3. Install dependencies (takes 5-10 minutes)
pip install -r requirements.txt

# 4. Prepare the data
python prepare_data.py

# 5. Run the analysis (takes 5-10 minutes)
python se_assignment1_1_code.py
```

### Expected Results

After running, you should see:
- Console output showing model performance
- 8 output files in the directory
- Final test set Brier score and accuracy
- Betting simulation results

### Naming Convention

Files follow the naming pattern: `se_assignment1_<group_number>_<type>`

- Replace `<group_number>` with your actual group number
- Current files use `1` as group number

To rename for your group (e.g., group 5):
```bash
# Python
mv se_assignment1_1_code.py se_assignment1_5_code.py
mv se_assignment1_1_data.csv se_assignment1_5_data.csv

# Update references in prepare_data.py:
# Change output_file = 'se_assignment1_1_data.csv'
# To:     output_file = 'se_assignment1_5_data.csv'

# Update references in se_assignment1_5_code.py:
# Change df = pd.read_csv('se_assignment1_1_data.csv')
# To:     df = pd.read_csv('se_assignment1_5_data.csv')
```

### Troubleshooting

**Problem:** Can't find data files
- Solution: Make sure `gamedata.csv` and `playergamedata.csv` are in the same directory

**Problem:** Package installation fails
- Solution: Upgrade pip first: `pip install --upgrade pip`

**Problem:** Script crashes
- Solution: Check you have at least 4GB RAM available
